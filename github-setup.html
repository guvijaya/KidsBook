🐙 GitHub Setup Guide for Kids Essays Website

Host your markdown essays on GitHub for free and edit them directly in your browser!

✨ Why GitHub is Perfect for This Project:

  • 🆓 Completely Free - No hosting costs
  • ✏️ Easy Editing - Edit files directly in GitHub's web interface
  • 📱 Edit Anywhere - Update content from any device with internet
  • 🔄 Version Control - Keep track of all changes and revert if needed
  • 👥 Collaboration - Multiple people can contribute essays
  • 🚀 Instant Publishing - Changes appear immediately on your website
  • 🔒 Backup - Your content is safely stored in the cloud

Step 1: Create a GitHub Repository

  • Go to GitHub.com and sign in (or create an account)
  • Click the "New" button or "+" icon → "New repository"
  • Repository name: kids-essays (or any name you prefer)
  • Description: Markdown essays for kids website
  • Make sure it's set to Public (required for free hosting)
  • Check "Add a README file"
  • Click "Create repository"
Create New Repository

Step 2: Create the Essays Folder Structure

  1. In your new repository, click "Create new file"
  2. Type: essays/en/my-family.md (GitHub will create the folders automatically)
  3. Add some sample content (see example below)
  4. Click "Commit new file"
  5. Repeat for other languages and essays

Recommended folder structure:

essays/ ├── en/ │ ├── my-family.md │ ├── my-pet.md │ ├── school-day.md │ ├── favorite-food.md │ └── summer-vacation.md ├── es/ │ ├── mi-familia.md │ ├── mi-mascota.md │ └── (other Spanish essays) ├── fr/ │ ├── ma-famille.md │ └── (other French essays) └── (other language folders)

Step 3: Configure Your Website

Edit js/github-content.js in your website and update these lines:

// Replace these with your actual GitHub details: this.githubUsername = 'YOUR_GITHUB_USERNAME'; this.repositoryName = 'kids-essays'; // or your repo name this.branch = 'main'; // or 'master' if that's your default

Example:

this.githubUsername = 'johndoe'; this.repositoryName = 'kids-essays'; this.branch = 'main';

Step 4: Add Your First Essay

Click "Create new file" in your repository and create essays/en/my-family.md:

# My Family My family is the most important thing in my life! 👨‍👩‍👧‍👦 ## Who's in My Family? I live with my **mom**, **dad**, and my little sister Sarah. We also have a golden retriever named Max! ### What We Love to Do Together - 🎮 Play board games on Sunday evenings - 🍳 Cook pancakes together on Saturday mornings - 🚗 Take road trips during school holidays - 📚 Read bedtime stories ## Why I Love My Family My family always supports me and makes me laugh. When I'm sad, they cheer me up. When I'm happy, they celebrate with me! *Family is where life begins and love never ends.* ❤️

Step 5: Test Your Setup

  • Save your changes to github-content.js
  • Refresh your website
  • Select a language and click "Open Book"
  • If configured correctly, you should see your GitHub essays!

🎉 Success! Now You Can:

  • Edit essays online: Go to your GitHub repository, click on any .md file, then click the pencil icon to edit
  • Add new essays: Click "Create new file" in your repository
  • See changes instantly: Your website will show updates immediately
  • Collaborate: Invite others to contribute to your repository

🚀 Quick Edit Workflow

  1. Go to your GitHub repository
  2. Navigate to the essay file you want to edit
  3. Click the pencil icon (✏️) to edit
  4. Make your changes in the editor
  5. Scroll down and click "Commit changes"
  6. Your website will show the updates immediately!

📝 Markdown Tips for Kids Essays:

  • # Title - Main heading
  • ## Section - Section heading
  • **bold text** - Make text bold
  • *italic text* - Make text italic
  • - item - Bullet point
  • 1. item - Numbered list
  • Add emojis to make it fun! 🎉📚✨

📱 Mobile Editing

You can edit your essays from your phone too!

  • Install the GitHub mobile app
  • Or use your mobile browser to visit github.com
  • Navigate to your file and tap the edit button
  • Edit and commit changes just like on desktop
