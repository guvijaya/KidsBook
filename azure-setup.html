🔧 Azure Blob Storage Setup Guide

Follow these steps to configure Azure Blob Storage for your Kids Essays Website.

Step 1: Create Azure Storage Account

  • Log into Azure Portal
  • Click "Create a resource" → "Storage account"
  • Choose your subscription and resource group
  • Enter a unique storage account name (remember this!)
  • Select your region
  • Choose "Standard" performance and "LRS" redundancy
  • Click "Review + Create" then "Create"

Step 2: Create Blob Container

  • Go to your new storage account
  • In the left menu, click "Containers"
  • Click "+ Container"
  • Name: essays
  • Public access level: Blob (anonymous read access for blobs only)
  • Click "Create"

Step 3: Upload Sample Content

Create this folder structure in your container:

essays/ ├── en/ │ ├── my-family.md │ ├── my-pet.md │ ├── school-day.md │ ├── favorite-food.md │ └── summer-vacation.md ├── es/ │ ├── mi-familia.md │ └── (other Spanish essays) └── (other language folders)

Step 4: Configure the Website

Edit js/azure-storage.js and replace:

this.storageAccountName = 'your-storage-account-name';

with your actual storage account name:

this.storageAccountName = 'mystorageaccount123';

Step 5: Test Your Setup

  • Open your website in a browser
  • Select a language
  • Click "Open Book"
  • If configured correctly, you should see your essays!

⚠️ CORS Configuration

If you encounter CORS errors, you may need to configure CORS settings in your storage account:

  • Go to Storage Account → Settings → Resource sharing (CORS)
  • Add a new CORS rule for Blob service:
  • Allowed origins: * (or your domain)
  • Allowed methods: GET
  • Allowed headers: *
  • Max age: 86400

✅ Success!

Once configured, your website will load essays from Azure Blob Storage. The sample content will be replaced with your actual markdown files.

Need Help?

Check the console in your browser's developer tools for debugging information. The website will show sample content until Azure is properly configured.
