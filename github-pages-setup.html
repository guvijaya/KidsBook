Step 1: Repository Setup

Your website code and essays will all be in one single repository.

  1. Go to GitHub.com and sign in
  2. Click "New" or "+" â†’ "New repository"
  3. Repository name: kids-essays-website (or your preferred name)

Step 2: Upload Your Website Files

Upload all the website files to your repository:

  1. In your repository, click "uploading an existing file" or drag & drop
  2. Upload these files and folders:
    • index.html
    • css/ folder (with styles.css)
    • js/ folder (with app.js and github-content.js)
    • essays/ folder (with your markdown files)
  3. Commit the files with message: "Initial website upload"

Step 3: Configure GitHub Details

Update the configuration to match your repository:

  1. In your repository, navigate to js/github-content.js
  2. Click the pencil icon (âœï¸) to edit
  3. Update these lines:

Click "Commit changes" when done.

Step 4: Enable GitHub Pages

This is where the magic happens - make your website live!

  1. In your repository, go to Settings (top menu)
  2. Scroll down to "Pages" in the left sidebar
  3. Under "Source":
    • Select: Deploy from a branch
    • Branch: main (or your default branch)
    • Folder: / (root)
  4. Click "Save"
  5. GitHub will show you your website URL: https://yourusername.github.io/repository-name
